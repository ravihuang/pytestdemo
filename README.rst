python自动化测试工程
=======================
HTTP REST接口及Selenium,Appium all-in-one自动化测试框架

目录说明
--------
- tests/config.py: 配置参数
- tests/ui: Selenium测试脚本
- tests/if: 北向接口测试脚本
- tests/stub: 南向接口测试脚本
- tests/android: appium for android
- tests/ios: appium fot ios
- lib：存放自定义库
- conf：存放配置文件
- testdata: 存放测试数据
